
## Description

CoinMarket is a website built with Nestjs. It displays cryptocurrency prices using coinmarketcap.com API.

## Installation

```bash
$ npm install
```

## Create .env file in the root folder

```bash

API_KEY="YOUR-COINBASE-API-KEY"

```

## Running the app
```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
